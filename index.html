SynEpis - Synthetic Epistemology

This project was intended to produce a monograph on "synthetic epistemology" as conceived by the author (but distinct from prior usage of that term). The project is currently on hold, having been superceded by the SPaDE project of which "Synthetic Philosophy" (which extends synthetic epistemology) is an major part. The PDFs that represented work in progress are available here, at some future date it may be that progress with SPaDE enables a return to and advance in this project.
